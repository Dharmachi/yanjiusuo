# 嫣究所

立嫣的初二数理研究所——**能玩的概念，不是能刷的题**。

新人教版 2024 · 八年级上册 · 数学 6 章 + 物理 6 章 · 13 个模块 / 113 个知识点。

纯静态 PWA：vanilla JS，无构建、无依赖、无追踪。可离线使用、可添加到主屏。

## 本地运行

```bash
python3 -m http.server 8246
# 然后打开 http://localhost:8246
```

## 结构

- `js/kits/` — 共享引擎（几何、图像、光路、天平量筒、代数瓷砖、音频、自测、讲透卡…）
- `js/modules/` — 13 个章节模块
- `data/` — 各章自测题库
- `sw.js` — Service Worker（离线缓存；改动资源需 bump VERSION）
